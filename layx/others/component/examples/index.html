Layx Component System

Below are examples demonstrating different features of the component system.

1. Static Component

A simple component with static HTML content.

2. Dynamic Component & Props

Passing properties to a component. prop="'John Doe', 'Engineer'"

Using global context variable. prop="globalAdmin, 'System Administrator'"

3. Nested Components

A component that contains other components.

4. Object Props

Passing a single object as props. prop="{ title: 'Notification', message: 'This is an object prop example.' }"

5. Render Table from List

Rendering a table by mapping over a list of objects (userList) defined in the global context.

6. Prop Inheritance

Demonstrating how a child component inherits context variables (like themeColor) from its parent automatically.
